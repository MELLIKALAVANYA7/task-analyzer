# Task Analyzer

Task Analyzer is a web-based application designed to help users manage and track their tasks efficiently. The system allows users to create, update, and monitor tasks, categorize them, and prioritize based on importance or deadlines. It also provides insights into task completion trends for better productivity management.

## Features

- Add, edit, and delete tasks
- Categorize tasks by type or project
- Set priorities and deadlines
- Track completed vs pending tasks
- User-friendly interface

## Tech Stack

- **Backend:** Python, Django  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** SQLite / PostgreSQL (can be configured)

## Getting Started

1. Clone the repository  
2. Install dependencies (Python, Django)  
3. Run the server and start managing your tasks
